Spice Store

This is a Next.js website for Habakeeat, a Syrian spice store based in Drikeish, Tartous. The site presents handcrafted spice blends, pure single spices, traditional recipe ideas, and brand information in a polished storefront layout.

Main features

Responsive landing page with a hero section, product sections, recipes, about section, and footer.

English and Arabic language switching with left to right and right to left layout support.

Light and dark theme switching.

Animated sections and hover effects using Framer Motion.

WhatsApp links for ordering and recipe requests.

Product image assets for spice blends, pure spices, and recipe cards.

Tech stack

Next.js 16
React 19
TypeScript
Tailwind CSS
Radix UI components
Framer Motion
Lucide React icons

Project structure

app contains the main Next.js pages and layout.

components contains the storefront sections and reusable UI components.

lib contains shared utilities and context providers for language and theme.

public contains images, icons, and placeholder assets.

styles contains global styling.

Getting started

Install dependencies with npm install.

Run the development server with npm run dev.

Build for production with npm run build.

Start the production server with npm run start.

Lint the project with npm run lint.

Notes

The project is private and currently uses local static product data inside React components. Product updates can be made by editing the arrays in the related section components.
